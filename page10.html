Creating my website

As our first coding challenge we had to create our own website from github using the MDEF template provided to us. This website will be used for showing all my personal progress, my notes, struggles and projects throughout the year.

Software's downloaded for creation of github site

- Gitbash
- Mkdocs
- Mobirise
- VS code
- Python 3

Damn it's not working

At the beginning I really didn't know what I was doing. By looking up a lot of things on internet, watching Josep's video and using chat GPT, I managed to create my Github website from the MDEF template. I modified the colours and text in mkdocs and visual studio. From there, I created my website in mobirise and connected it to github. This is my solution for now until I completely understand how to modify the original mdef template by code. Another option would be trying to code within mobirise to do specific stuff that I can't do within the site.

Josep's link https://fablabbcn-projects.gitlab.io/learning/educational-docs/mdef/
